# Jupyter Demo Project

Demonstrates best practices for using Jupyter notebooks in a uv-managed environment.

## Quick Start

```bash
# One-time setup
./setup.sh

# Or manually:
uv sync
source .venv/bin/activate
jupyter lab
```

## Project Structure

```
/samples/jupyter-demo/
├── notebooks/           # All notebooks (.ipynb and .py pairs)
│   ├── 00_basic_panda.ipynb
│   ├── 00_basic_panda.py        # Simple notebook to make sure everything works
│   ├── 01_data_analysis.ipynb
│   ├── 01_data_analysis.py      # demonstrates use of ML libraries and interaction with code in /src
├── src/                 # Reusable Python modules
│   ├── data_utils.py    # Data loading and processing
│   └── notebook_converter.py  # Format conversion utilities
├── data/                # Data files
│   └── sample.csv       # generated by setup.sh
├── outputs/             # Generated outputs
│   ├── figures/         # Saved plots
│   └── reports/         # Converted notebooks
├── setup.sh             # install, setup, generate sample data and notebook
└── pyproject.toml       # Project dependencies
```

## Working with Notebooks

### Start Jupyter Lab

```bash
source .venv/bin/activate
jupyter lab --no-browser --port=8888
```

### Convert Notebooks

```bash
# To Python script
jupyter nbconvert --to script notebooks/01_data_analysis.ipynb

# To HTML report
jupyter nbconvert --to html notebooks/01_data_analysis.ipynb --output-dir outputs/reports/

# To Markdown
jupyter nbconvert --to markdown notebooks/01_data_analysis.ipynb
```
